# **¿Qué es Java?** 

<img class="hola" src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/java/java-original.svg" alt="java logo"/>

Java es un lenguaje de programación popular, creado en 1995.

Es propiedad de Oracle y más de **3.000 millones** de dispositivos ejecutan Java.

- Es utilizado para:
    1. Aplicaciones móviles
    2. Aplicaciones de escritorio
    3. Aplicaciones web
    4. Servidores web y servidores de aplicaciones
    5. Juegos
    6. Conexión a la base de datos

> Entre muchas otras aplicaciones.

## **¿Por qué usar Java?**

- Java funciona en diferentes plataformas (Windows, Mac, Linux, Raspberry Pi, etc.)
- Es uno de los lenguajes de programación más populares del mundo
- Tiene una gran demanda en el mercado laboral actual
- Es fácil de aprender y simple de usar
- Es de código abierto y gratuito
- Es seguro, rápido y potente
- Tiene un gran apoyo de la comunidad (decenas de millones de desarrolladores)
- Java es un lenguaje orientado a objetos que da una estructura clara a los programas y permite reutilizar el código, reduciendo los costes de desarrollo
- Como Java está cerca de C++ y C#, facilita a los programadores el cambio a Java o viceversa
