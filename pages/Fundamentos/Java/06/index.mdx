
## **Tipos de datos Java**


```java
int myNum = 5;               // Integer (whole number)
float myFloatNum = 5.99f;    // Floating point number
char myLetter = 'D';         // Character
boolean myBool = true;       // Boolean
String myText = "Hello";     // String

```

Los tipos de datos se dividen en dos grupos:

- Tipos de datos primitivos:`byte`, `short`, `int`, `long`, `float`, `double`, `boolean`y `char`
- Tipos de datos no primitivos, como, Matrices y Clases:`[String](<https://www.w3schools.com/java/java_strings.asp>)`
