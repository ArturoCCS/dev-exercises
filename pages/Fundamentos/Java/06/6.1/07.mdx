
### **Strings**

El tipo de datos se utiliza para almacenar una secuencia de caracteres (texto). Los valores de cadena deben ir entre comillas dobles:`String`

```java
String greeting = "Hello World";
System.out.println(greeting);
```

> El tipo String es tan utilizado e integrado en Java, que algunos lo llaman "el **noveno** tipo especial".

Una cadena en Java es en realidad un tipo **de datos no primitivo**, porque se refiere a un objeto. El objeto String tiene métodos que se utilizan para realizar determinadas Operaciones en cadenas. **No te preocupes si aún no entiendes el término "objeto".**
