
### **Caracteres**

El tipo de datos se utiliza para almacenar un **solo** carácter. `char` el carácter debe ser entre comillas simples, como 'A' o 'c':

```java
char myGrade = 'B';
System.out.println(myGrade);
```

Alternativamente, si está familiarizado con los valores ASCII, puede usarlos para mostrar ciertos caracteres:

```java
char myVar1 = 65, myVar2 = 66, myVar3 = 67;
System.out.println(myVar1);
System.out.println(myVar2);
System.out.println(myVar3);
```
