
### **Tipos enteros**

**Byte**

El tipo de datos puede almacenar números enteros de -128 a 127. Esto se puede usar en lugar de otros tipos enteros para que se guarde en la memoria cuando esté seguro de que el valor estará entre -128 y 127: `byte`

```java
byte myNum = 100;
System.out.println(myNum);
```

Short

El tipo de datos puede almacenar números enteros de -32768 a 32767:`short`

```java
short myNum = 5000;
System.out.println(myNum)
```

Int

El tipo de datos puede almacenar números enteros de -2147483648 a 2147483647. El tipo de datos es `inte`, este es el tipo de dato preferido cuando creamos variables con un valor numérico.

```java
int myNum = 100000;
System.out.println(myNum);
```

Long

El tipo de datos puede almacenar números enteros de -9223372036854775808 a 9223372036854775807. Esto se usa cuando int no es lo suficientemente grande como para almacenar el valor. Tenga en cuenta que debe terminar el valor con una "L":`long`

```java
long myNum = 15000000000L;
System.out.println(myNum);
```
