## **Comentarios en Java**

Los comentarios se pueden utilizar para explicar el código Java y hacerlo más legible. También se puede utilizar para: Evite la ejecución al probar código alternativo.

### **Comentarios de una sola línea**

Los comentarios de una sola línea comienzan con dos barras diagonales ().`//`

Cualquier texto entre y al final de la línea es ignorado por Java (no se ejecutará).`//`

En este ejemplo se usa un comentario de una sola línea antes de una línea de código:

**Ejemplo:**

```java
// Este es un comentario
System.out.println("Hola Mundo");
```

En este ejemplo se usa un comentario de una sola línea al final de una línea de código:

```java

System.out.println("Hola Mundo");// Este es un comentario
```

### **Comentarios de varias líneas de Java**

Los comentarios de varias líneas comienzan con .`/**/`

Cualquier texto entre y será ignorado por Java.`/**/`

En este ejemplo se usa un comentario de varias líneas (un bloque de comentarios) para explicar el código:

**Ejemplo:**

```java
/* The code below will print the words Hello World
to the screen, and it is amazing */
System.out.println("Hello World");
```

💡 ¿Comentarios de una o varias líneas?

Depende de ti cuál quieres usar. Normalmente, lo usamos para comentarios cortos y durante más tiempo.`///* */`
