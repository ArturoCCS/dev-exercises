
### **Declarar Múltiples Variables**

Para declarar más de una variable del **mismo tipo**, puede Utilice una lista separada por comas:

En lugar de escribir:

```java
int x = 5;
int y = 6;
int z = 50;
System.out.println(x + y + z);
```

Simplemente puedes escribir:

```java
int x = 5, y = 6, z = 50;
System.out.println(x + y + z);
```

### **Un valor para varias variables**

También puede asignar el **mismo valor** a varias variables en una línea:

```java
int x, y, z;
x = y = z = 50;
System.out.println(x + y + z);
```
