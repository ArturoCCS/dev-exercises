
### **Mostrar variables**

El método se utiliza a menudo para mostrar variables.`println()`

Para combinar texto y una variable, utilice el carácter:`+`

```java
String name = "John";
System.out.println("Hello " + name);
```

También puede usar el carácter para agregar una variable a otra variable:`+`

```java
String firstName = "John ";
String lastName = "Doe";
String fullName = firstName + lastName;
System.out.println(fullName);
```

Para valores numéricos, el carácter funciona como un [operador](https://www.w3schools.com/java/java_operators.asp) matemático (tenga en cuenta que aquí usamos variables (enteras):`+int`

```java
int x = 5;
int y = 6;
System.out.println(x + y); // Print the value of x + y
```

A partir del ejemplo anterior, puede esperar:

- x almacena el valor 5
- y almacena el valor 6
- Luego usamos el método para mostrar el valor de x + y, que es **11** `println()`
