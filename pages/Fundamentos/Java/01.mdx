
## **Sintaxis de Java**

### **El método principal**

El método es obligatorio y usted verlo en todos los Java programa:`main()`

**Ejemplo:**

```java
public static void main(String[] args)
```

### System.out.println()

Dentro del método, podemos usar el método para imprimir una línea de texto en la pantalla:`main()println()`

**Ejemplo:**

```java
public static void main(String[] args) {
	System.out.println("Hello World");
}
```

- **Nota :** Las llaves marcan el principio y el final de un bloque de código.`{}`
    
    `System` es una clase Java incorporada que contiene miembros útiles, como , que es la abreviatura de "salida". El método, abreviatura de "Imprimir línea", se utiliza para imprimir un valor en la pantalla (o un archivo).`outprintln()`
    
    No te preocupes demasiado por , y . Solo debes saber que los necesitas juntos para imprimir cosas en la pantalla.`Systemoutprintln()`
    
    También debe tener en cuenta que cada instrucción de código debe terminar con un punto y coma ().`;`
