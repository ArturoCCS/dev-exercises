
## **Salida**

- **Imprimir texto**
    
    Puede agregar tantos métodos como desee. Tenga en cuenta que agregará una nueva línea para cada método:`println()`
    
    **Ejemplo:**
    
    ```java
    System.out.println("Hello World!");
    System.out.println("I am learning Java.");
    System.out.println("It is awesome!");
    ```
    
    ### **Comillas dobles**
    
    Cuando se trabaja con texto, debe estar entre comillas dobles.`""`
    
    Si olvida las comillas dobles, se produce un error:
    
    **Ejemplo:**
    
    ```java
    ✅ System.out.println("This sentence will work!");
    ```
    
    ```java
    ❌ System.out.println(This sentence will produce an error);
    ```
    
    ### **El método print()**
    
    También hay un método, que es similar a .`print()println()`
    
    La única diferencia es que no inserta una nueva línea al final de la salida:
    
    **Ejemplo:**
    
    ```java
    System.out.print("Hello World! ");
    System.out.println("I will print on the same line.");
    ```
    
- **Imprimir números**
    
    También puede utilizar el método para Imprime números.`println()`
    
    Sin embargo, a diferencia del texto, no ponemos números dentro de dobles presupuestos:
    
    **Ejemplos:**
    
    ```java
    System.out.println(3);
    System.out.println(358);
    System.out.println(50000);
    ```
    
    También puede realizar cálculos matemáticos dentro del método:`println()`
    
    ```java
    System.out.println(3 + 3);
    
    System.out.println(2 * 5);
    ```
